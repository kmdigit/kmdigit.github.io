파이썬 :: PIP 오프라인 설치

whl 파일 다운로드

pip 접속이 가능한 온라인 환경이라면 pip의 download명령어로 필요한 파일을 다운로드 한다.

-d옵션으로 다운로드 받을 폴더를 지정할 수 있으니 필요한 곳을 지정하고 원하는 패키지를 입력하면 의존성 있는 모든 패키지가 다운 받아 진다.

오프라인 설치

requirements.txt로 설치가 가능하지만 다운로드 받을 때처럼 패키지 명으로도 설치할 수 있다.

--no-index옵션은 패키지 인덱스를 무시하고 오직 -f, --find-links옵션으로 지정한 URL로 접근하여 다운로드 및 설치 하려는 옵션이다. 즉 —no-index와 -f 옵션은 항상 같이 다닌다.

-f, --find-links옵션으로 현재 디렉토리를 지정했으므로 pip는 현재 디렉토리에서 flask에 필요한 의존성 패키지들을 찾아서 설치한다. 이미 온라인 환경에서 필요한 패키지를 모두 다운 받았으므로 해당 파일을 복사한 디렉토리로 지정 해주면 알맞게 설치가 가능하다.
